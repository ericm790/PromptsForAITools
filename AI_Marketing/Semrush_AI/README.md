# Semrush AI — Prompt Guide

## Overview
Semrush is a comprehensive digital marketing intelligence platform covering SEO, PPC, content, social, and competitive analysis. Its AI features enhance data-driven marketing decisions.

## How to Use

### System Prompt
1. `System_Prompt.txt` provides the strategic framework for using Semrush's AI capabilities.
2. Covers audit methodologies, competitive analysis, and content strategy.

### User Prompts
- Use in Semrush's **ContentShake AI**, **SEO Writing Assistant**, or as strategic briefs.
- Always specify your industry, target audience, and business goals.

## Tips for Best Results
- **Start with competitors** — competitor analysis reveals the biggest opportunities.
- **Use keyword clustering** — group keywords by topic, not just individual terms.
- **Prioritize by impact** — focus on high-volume, low-difficulty keywords first.
- **Combine channels** — use SEO + PPC data together for complete keyword intelligence.
- **Monitor regularly** — set up position tracking and site audit schedules.
