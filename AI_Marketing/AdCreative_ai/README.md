# AdCreative.ai — Prompt Guide

## Overview
AdCreative.ai generates high-performing ad creatives with AI-powered performance prediction. It optimizes for conversion across Meta, Google, LinkedIn, TikTok, and more.

## How to Use

### System Prompt
1. `System_Prompt.txt` provides the ad creative strategy framework.
2. Covers platform-specific optimization, creative frameworks, and performance prediction.

### User Prompts
- Use in AdCreative.ai's **creative generation** or **text generation** tools.
- Specify platform, objective, audience, and brand guidelines.

## Tips for Best Results
- **Specify the platform** — each platform has unique specs and norms.
- **Include your value props** — give the AI clear selling points.
- **Request multiple variations** — always A/B test creatives.
- **Set campaign objectives** — awareness vs. conversion drives different creative styles.
- **Use performance predictions** — prioritize top-ranked creatives for initial spend.
