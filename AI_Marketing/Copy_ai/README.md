# Copy.ai — Prompt Guide

## Overview
Copy.ai is an AI copywriting platform focused on sales outreach, marketing copy, and workflow automation. It excels at high-converting short-form and long-form content.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` as the foundation for Copy.ai workflows and chat instructions.
2. Establishes copywriting frameworks, sales methodology, and channel-specific optimization.

### User Prompts
- Use in Copy.ai's **Chat**, **Workflow builder**, or **Document editor**.
- Provide target audience, value propositions, and desired tone.

## Tips for Best Results
- **Specify the audience** — include industry, role, company size, and pain points.
- **Use frameworks** — reference AIDA, PAS, or BAB in your prompt for structured output.
- **Request variations** — ask for multiple versions for A/B testing.
- **Include personalization** — use {{variables}} for scalable personalized content.
- **Define the channel** — email, LinkedIn, Google Ads, etc. have different best practices.
