# Surfer SEO — Prompt Guide

## Overview
Surfer SEO is an AI-powered content optimization platform that analyzes top-ranking pages to provide data-driven content guidelines for SEO ranking improvement.

## How to Use

### System Prompt
1. `System_Prompt.txt` provides the SEO optimization framework and methodology.
2. Use it to inform content briefs, optimization audits, and keyword strategies.

### User Prompts
- Use in Surfer SEO's **Content Editor**, **SERP Analyzer**, or **Content Planner**.
- Always start with your target keyword and search intent.

## Tips for Best Results
- **Start with keyword research** — use Surfer's keyword suggestions for best targets.
- **Follow content scores** — aim for 80+ content score against top competitors.
- **Don't keyword stuff** — integrate NLP terms naturally.
- **Update regularly** — re-audit content quarterly for ranking maintenance.
- **Use content briefs** before writing — structured content ranks better.
