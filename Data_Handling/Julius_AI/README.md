# Julius AI — Prompt Guide

## Overview
Julius AI is a conversational data analysis platform that lets users upload datasets and ask questions in natural language. It performs statistical analysis, creates visualizations, builds predictive models, and generates reports — all without requiring the user to write code. Julius supports CSV, Excel, Google Sheets, and database connections.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` to configure AI behavior in data analysis chatbots or internal analytics tools.
2. Adapt the prompt for any no-code data analysis platform or LLM-powered analytics workflow.

### User Prompts
- Upload your dataset first, then use these prompts as templates.
- Always mention your column names and data types for precise analysis.
- Specify the output format you need (charts, tables, reports, recommendations).

## Tips for Best Results
- **Upload clean data** — remove obvious junk rows before uploading for better results.
- **Describe your business context** — the more context you give, the more relevant the insights.
- **Ask iteratively** — start with an overview, then drill into specific areas of interest.
- **Request visualizations** — explicitly ask for charts and graphs to accompany numerical results.
- **Ask for recommendations** — don't just ask "what happened" — ask "what should we do about it."
