# Obviously AI — Prompt Guide

## Overview
Obviously AI is a no-code machine learning platform that lets users build predictive models by simply asking a question about their data. It automates the entire ML pipeline — data cleaning, feature engineering, model selection, training, evaluation, and deployment — making AI accessible to business users without any coding or data science knowledge.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` to configure AI behavior in prediction-focused tools or internal analytics platforms.
2. Adapt it for any no-code ML workflow where business users need guided model building.

### User Prompts
- Upload your data and clearly state what you want to predict.
- Frame your question as "Predict [target] based on [available data]."
- Include business context so predictions are tied to actionable outcomes.

## Tips for Best Results
- **State your prediction target clearly** — be explicit about which column you want to predict.
- **Include enough historical data** — more rows generally means better predictions (aim for 500+).
- **Describe your use case** — how you'll act on the predictions determines the right model.
- **Ask for explanations** — always request feature importance and reasoning, not just predictions.
- **Request ROI estimates** — ask how much money the predictions could save or generate.
