# DALL-E 3 — Prompt Guide

## Overview
DALL-E 3 is OpenAI's advanced image generation model, integrated into ChatGPT. It excels at prompt fidelity, creative interpretation, and conversational image generation.

## How to Use

### System Prompt
1. `System_Prompt.txt` provides quality guidelines and prompt structure for DALL-E 3.
2. Use the prompt guidelines to craft effective image descriptions.

### User Prompts
- In **ChatGPT**, simply describe what you want and DALL-E 3 generates it.
- Via the **OpenAI API**, send prompts programmatically with size and quality options.

## Tips for Best Results
- **Be descriptive** — include subject, setting, style, mood, and lighting.
- **Conversational refinement** — in ChatGPT, iterate through conversation.
- **Specify size** — 1024x1024 (square), 1792x1024 (landscape), 1024x1792 (portrait).
- **Use "hd" quality** for detailed, professional outputs.
- **Natural style** for realism, **vivid** for dramatic/artistic.
