# Midjourney — Prompt Guide

## Overview
Midjourney is the leading AI image generator known for exceptionally aesthetic and artistic outputs. It operates through Discord and a web interface.

## How to Use

### System Prompt
1. `System_Prompt.txt` provides the quality framework and prompt construction guide.
2. Use it as a mental model for crafting Midjourney prompts — not as a direct system instruction.

### User Prompts
- Enter prompts in the Midjourney Discord bot (`/imagine`) or at **midjourney.com**.
- Each prompt follows the pattern: **Subject + Style + Lighting + Mood + Parameters**.
- Append `--ar`, `--s`, `--v`, and other parameters at the end.

## Tips for Best Results
- **Be specific about lighting** — "golden hour," "volumetric fog," "neon backlight" dramatically change results.
- **Reference art styles** — mention artists, movements, or mediums for style guidance.
- **Use `--s` (stylize)** — higher values (500-1000) for artistic, lower (100-250) for literal.
- **Use `--style raw`** for photorealistic, less stylized outputs.
- **Negative prompts** — use `--no` to exclude unwanted elements.
- **Iterate with variations** — use V1-V4 buttons to explore directions.
