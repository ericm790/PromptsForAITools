# Flux.1 — Prompt Guide

## Overview
Flux.1 by Black Forest Labs is a state-of-the-art open-source image generator known for exceptional prompt adherence and text rendering capabilities.

## How to Use

### System Prompt
1. `System_Prompt.txt` provides the quality framework for Flux.1 image generation.
2. Understand the model variants (Pro, Dev, Schnell) and choose based on your needs.

### User Prompts
- Use Flux.1 via **Replicate**, **fal.ai**, **ComfyUI**, or self-hosted.
- Write natural language prompts — no special syntax needed.
- Put text you want rendered in quotation marks.

## Tips for Best Results
- **Use natural language** — Flux.1 understands complex descriptions without special syntax.
- **Be specific about quantities** — "exactly three" instead of "some."
- **Quote text** — put text to render in images inside quotation marks.
- **Choose the right variant** — Pro for quality, Schnell for speed, Dev for experimentation.
- **Describe lighting specifically** — "golden hour," "overcast," "studio softbox from left."
