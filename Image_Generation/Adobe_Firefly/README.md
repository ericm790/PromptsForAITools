# Adobe Firefly — Prompt Guide

## Overview
Adobe Firefly is Adobe's commercially safe AI image generator, trained on Adobe Stock and licensed content. It's integrated into Photoshop, Illustrator, and Express.

## How to Use

### System Prompt
1. `System_Prompt.txt` provides the framework for professional creative generation with Firefly.
2. Emphasizes commercial safety, brand consistency, and Adobe workflow integration.

### User Prompts
- Use at **firefly.adobe.com** or within **Photoshop, Illustrator, Express**.
- Select Content Type (Photo, Illustration, Art, Graphic) for style guidance.
- Upload style and structure references for consistency.

## Tips for Best Results
- **Commercially safe** — all Firefly outputs are cleared for commercial use.
- **Use references** — upload style images to match your brand aesthetic.
- **Specify Content Type** — Photo, Illustration, Art, or Graphic controls the output style.
- **Integrate with Photoshop** — use Generative Fill/Expand for editing workflows.
- **Content Credentials** — all outputs include metadata showing AI generation.
