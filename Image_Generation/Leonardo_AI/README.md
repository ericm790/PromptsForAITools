# Leonardo AI — Prompt Guide

## Overview
Leonardo AI is a professional image generation platform with multiple fine-tuned models, ControlNet, and canvas editing. It excels at game assets, concept art, and commercial creative work.

## How to Use

### System Prompt
1. `System_Prompt.txt` provides the framework for professional image generation with Leonardo AI.
2. Use it as a guide for model selection, prompt construction, and quality standards.

### User Prompts
- Enter prompts at **leonardo.ai** in the Image Generation tool.
- Select the appropriate model for your use case.
- Use both positive and negative prompts for best results.

## Tips for Best Results
- **Choose the right model** — RPG Model for fantasy, Kino XL for photos, Anime XL for anime.
- **Use negative prompts** — always exclude common artifacts (blurry, bad anatomy, extra fingers).
- **Use ControlNet** for precise poses and compositions.
- **Generate at higher resolution** for professional use.
- **Use the canvas editor** for inpainting and extending images.
