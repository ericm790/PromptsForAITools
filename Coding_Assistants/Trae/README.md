# Trae — Prompt Guide

## Overview
Trae is an AI-native IDE with multi-model support and a "Builder" mode for generating complete applications from prompts. It combines intelligent code completion with deep project understanding.

## How to Use

### System Prompt
1. Paste `System_Prompt.txt` into Trae's AI configuration settings.
2. This establishes behavior for both inline assistance and Builder mode.

### User Prompts
- Use in Trae's **chat panel** for conversational assistance.
- Use in **Builder mode** for full project generation.
- Specify your tech stack and requirements explicitly.

## Tips for Best Results
- **Builder mode** works best with detailed descriptions — include features, styling, and behavior.
- **Reference files** by name or path when asking about specific code.
- **Use different complexity levels** — simple completions vs. full architecture guidance.
- **Iterate** — refine generated code through follow-up conversations.
