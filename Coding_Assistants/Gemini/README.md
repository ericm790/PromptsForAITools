# Gemini — Prompt Guide

## Overview
Gemini is Google's most capable AI model with multimodal abilities (text, images, code, data). It excels at analytical, creative, and coding tasks.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` in Gemini's system instructions (API) or as preamble in conversations.
2. Optimizes for multimodal, analytical, and creative excellence.

### User Prompts
- Use at **gemini.google.com**, in Google AI Studio, or via the Gemini API.
- Upload images, documents, and code alongside text prompts.

## Tips for Best Results
- **Upload images/docs** for multimodal analysis — Gemini excels at visual understanding.
- **Ask for step-by-step reasoning** on complex problems.
- **Specify output format** — tables, code, reports, etc.
- **Use extended thinking** for hard problems — ask Gemini to "think step by step."
- **Iterate** — refine outputs through conversational follow-ups.
