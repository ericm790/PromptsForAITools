# Windsurf — Prompt Guide

## Overview
Windsurf is an AI-powered IDE that uses "Cascade" — a multi-step reasoning engine — to understand your full codebase and make intelligent, context-aware changes across multiple files.

## How to Use

### System Prompt
1. Paste `System_Prompt.txt` into Windsurf's AI configuration or use it as a session preamble.
2. This activates the Cascade reasoning methodology and quality standards.

### User Prompts
- Use these in Windsurf's **Chat panel** or **Cascade mode** for multi-file operations.
- Be specific about the scope of changes and reference existing project patterns.

## Tips for Best Results
- **Let Cascade work** — describe the end-goal and let Windsurf plan the cascade of changes.
- **Reference project patterns** — mention existing code/files that new code should follow.
- **Use for multi-file tasks** — Windsurf excels at changes that span many files.
- **Iterate quickly** — ask for modifications to generated code rather than re-describing the task.
- **Check diffs** — review Windsurf's proposed changes file-by-file before accepting.
