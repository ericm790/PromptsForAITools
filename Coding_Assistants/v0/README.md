# v0 — Prompt Guide

## Overview
v0 is Vercel's AI-powered generative UI tool that creates production-ready React components from text descriptions. It uses React, TypeScript, Tailwind CSS, and shadcn/ui.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` as a design quality baseline in v0 or similar generative UI tools.
2. It establishes standards for visual quality, accessibility, and code structure.

### User Prompts
- Enter prompts directly into v0's interface at **v0.dev**.
- **Be descriptive about visuals** — mention colors, spacing, animations, and layout.
- Each prompt generates a live, interactive preview.

## Tips for Best Results
- **Describe the visual design** — mention color schemes, themes, spacing, and typography.
- **Reference design systems** — "shadcn/ui style" or "Stripe's dashboard" for clarity.
- **Specify interactions** — hover effects, transitions, and state changes.
- **Include real content** — describe actual copy, not just structure.
- **Iterate visually** — use v0's edit feature to refine components.
