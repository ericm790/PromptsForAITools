# Bolt — Prompt Guide

## Overview
Bolt is an AI-powered platform that generates complete, runnable web applications from natural language descriptions. It creates full project structures with all code, configurations, and dependencies in one shot.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` as a quality reference for Bolt interactions.
2. Sets standards for code structure, UI quality, and completeness.

### User Prompts
- Enter prompts on **bolt.new** to generate complete applications.
- Describe the full application — features, design, interactions, and data.

## Tips for Best Results
- **Be comprehensive** — describe all features, pages, and interactions upfront.
- **Specify the design** — mention color themes, layout preferences, and UI style.
- **Include data models** — describe what data the app manages.
- **Mention interactivity** — specify animations, transitions, and dynamic behaviors.
- **Iterate** — use follow-up prompts to add features or fix issues.
