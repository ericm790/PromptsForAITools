# Replit — Prompt Guide

## Overview
Replit AI is a cloud-based coding assistant integrated into the Replit platform. It excels at rapid prototyping, full-stack development, and making coding accessible in a browser-based IDE.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` content as the preamble for Replit AI conversations.
2. This optimizes the AI for the Replit cloud environment and its specific features.

### User Prompts
- Use in Replit's **AI chat** for project generation and assistance.
- Be specific about the technology stack and desired features.

## Tips for Best Results
- **Start with the end goal** — describe the complete app you want to build.
- **Mention Replit-specific features** — Replit DB, Auth, Deployments — for seamless integration.
- **Ask for complete projects** — Replit AI excels at generating full, runnable applications.
- **Use iteratively** — start simple, then add features through conversation.
- **Check the console** — paste error messages for accurate debugging help.
