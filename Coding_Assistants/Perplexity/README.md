# Perplexity — Prompt Guide

## Overview
Perplexity is an AI-powered answer engine that combines real-time web search with AI reasoning. It provides sourced, up-to-date answers with inline citations.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` to set quality expectations for Perplexity-style interactions.
2. Emphasizes citation standards, multi-source synthesis, and structured responses.

### User Prompts
- Enter prompts at **perplexity.ai** or in the Perplexity app.
- Focus on research, comparison, and fact-finding queries.

## Tips for Best Results
- **Ask specific questions** — "What are the best X for Y in 2025?" outperforms vague queries.
- **Request comparisons** — Perplexity excels at multi-source comparative analysis.
- **Ask for sources** — explicitly request citations for higher accuracy.
- **Use Focus modes** — select Academic, Writing, or other modes for specialized results.
- **Follow up** — refine and deepen with follow-up questions.
