# ChatGPT — Prompt Guide

## Overview
ChatGPT by OpenAI is the world's most popular AI assistant, excelling at writing, coding, analysis, brainstorming, and research. It supports plugins, code interpreter, web browsing, and image generation.

## How to Use

### System Prompt
1. Use `System_Prompt.txt` as custom instructions in ChatGPT (Settings → Custom Instructions → System Prompt).
2. Or use it as a system message via the OpenAI API.

### User Prompts
- Use at **chat.openai.com** or in the ChatGPT app.
- Specify context, format, and audience for best results.

## Tips for Best Results
- **Set custom instructions** for consistent behavior across conversations.
- **Upload files** — ChatGPT can analyze CSVs, PDFs, images, and code files.
- **Use Code Interpreter** for data analysis, visualizations, and computations.
- **Request specific formats** — tables, bullet points, code, essays, etc.
- **Iterate** — refine outputs by giving specific feedback on what to change.
